Information


HOTISSUE BOYEOJUSHU는 유튜브의 인기 급상승 동영상을 바탕으로 인기 키워드 순위를 제공하는 웹사이트입니다.

인기 키워드 순위는 다음과 같은 방법으로 선정됩니다.

1. 높은 순위 영상의 제목에 포함된 단어
2. 여러 영상의 제목에서 언급된 단어

첫 화면의 rank 버튼을 누르면 한국의 현재 유튜브를 기준으로 상위 10개의 키워드가 보입니다.

사용방법은 다음과 같습니다.


메인 페이지의 국가_RANK 버튼을 클릭하면 선택한 국가 기준의 키워드 순위를 제공합니다.
COMPARE 버튼을 클릭하면 이전의 순위와 비교한 순위의 결과를 제공합니다.
단, COMPARE에서는 한국어 키워드만 제공합니다.
국가별 순위에서 키워드를 클릭하면, 유튜브 코리아에서 검색한 결과를 제공합니다.
